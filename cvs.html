Anonymous CVS

The whole of the GNU Cobol project is available via anonymous CVS. To access this, you will need the CVS application, available for free from Cyclic Software. After you have CVS installed, you will need to set your CVSROOT as follows (for Bourne-shell and variants): 

export CVSROOT=":pserver:cvs@acm.cs.umr.edu:/gnu-cobol"
or for csh-variants: 
setenv CVSROOT ":pserver:cvs@acm.cs.umr.edu:/gnu-cobol"

You will then need to issue the following commands: 

cvs login		(password: cvs)
cvs -z3 checkout cobol

This will leave you with a full source tree for the GNU Cobol project.
